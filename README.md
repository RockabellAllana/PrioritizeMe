# PrioritizeMe
Neurodivergent-centric task prioritizer
/*
The general vision is a simple program I can later add aesthetic value to
This program needs to be able to take a string input and save it as a call-able linked-list
This list will include due dates, and difficulty ratings from the user as well as an editable 
up to 150 character description of the assignment for clarity
Then, the program will take this list and set a priority value to each object
and organize itself to present the most impending assignment
We need:

A menu that allows the user to select from options: Add Assignment, List of Assignments, Change Priority, What should I work on next?, Exit
A function in the Add Assignment section that takes several inputs from the user to add as values to an organized list
A function under change priority to search for and alter values in the list
A function that lists assignments (automatically by priority as these values are organized this way)
A function that takes the due dates and difficulty ratings into account to keep the list in order of priority
A function that saves the data to an accessible .txt file in an organized, easy to read way until I learn how to 
write the program to ACTUALLY keep and store those values within it 

*/
